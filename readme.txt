## Description ##

Ultimate Redirect Manager is the most advanced redirection tool for WordPress. This Redirect tool will redirect your website's error links and URLs to the pages you select. You can also track and manage 404, 301, 302, and "Page Not Found" errors very easily. Reduce errors and raise your site's ranking with the aid of Ultimate Redirect Manager.

You can quickly and easily create and manage redirects without any prior knowledge of Apache or Nginx. Redirection can be used to redirect any URL if your WordPress site allows permalinks.

## Features: ##

**Custom 404 Page:** Choose a custom page to serve as your 404 error page.

**Manual Redirects:** Set up manual redirects for specific source URLs to redirect to destination URLs.

**Redirect Track:** Redirect URLs will be recorded in the WordPress database for tracking and monitoring redirect links in the plugin dashboard.

**User-Friendly Interface:** Manage redirect rules through a simple and intuitive settings interface in the WordPress admin dashboard.

## Usage: ##

**Setting Up Custom 404 Page:**

1. In the WordPress admin dashboard, go to Settings > Ultimate Redirect Manager.
2. Under the 404 Page Redirection section, select a page from the dropdown menu to serve as your custom 404 error page.
3. Click Save Changes.

**Adding Manual Redirects:**

1. In the WordPress admin dashboard, go to Settings > Ultimate Redirect Manager.
2. Under the Manual Redirect Table section, you can add manual redirects by providing the source URL, destination URL, and redirect type.
3. Click Save Rule to add the redirect rule.
